# Examples

Examples should become small, reproducible demonstrations with sanitized inputs, commands, outputs, and evaluation records.

## Planned Examples

- `math-proof-to-latex`: theorem statement to proof plan, verification notes, and polished LaTeX.
- `asymptotic-analysis`: recurrence or limit problem with Python-assisted checks.
- `web-game-ui-qa`: canvas/UI bug reproduction, patch, and screenshot evidence.
- `public-skill-distillation`: private incident to sanitized public `SKILL.md`.

Each example should include:

- input
- selected skills
- tool routing
- artifacts
- evaluation report
- known limitations
